# Support

## Getting Help

For usage questions, troubleshooting, and configuration help:

- Open a GitHub issue:
  - Bug reports: use the Bug Report template
  - Questions/enhancements: use the Feature Request template

## Before Opening an Issue

- Read [README.md](README.md)
- Check [docs/DEVELOPMENT.md](docs/DEVELOPMENT.md)
- Search existing issues for duplicates

## What to Include

- OS and runtime versions (Node, npm, Rust where relevant)
- Exact command(s) executed
- Error logs/output
- Steps to reproduce
